# HTML FPS Horror Game

A web-based first-person shooter game with a horror theme. Built using **HTML, CSS, and JavaScript**. 

## Features
- 3 interactive levels
- Unique skills per level
- Shooting effects with audio
- Basic FPS mechanics: movement, collision, and particle effects
- Uses free OBJ models and textures

## Tech Stack
- HTML / CSS / JavaScript
- OBJ 3D models
- Audio effects
